## How to Participate

All challenges are deployed live on the **Cyberium Arena** platform.

1. Log in to [Cyberium Arena](https://cyberium.io)
2. Navigate to **Cyber Grid → Arena**
3. Select a challenge and launch your instance
4. Read the briefing in this repo, analyze the target, extract the flag
5. Submit the flag in the format `CTF{...}` on the platform

Each challenge has **3 missions** to complete in order. The flag is the answer to Mission 3.
